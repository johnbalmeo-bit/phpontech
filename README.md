# Linamnam (phpontech)

A Filipino recipe sharing platform.

## Features
- User authentication
- Recipe posting, liking, saving, commenting
- Responsive frontend
- AJAX actions for interactivity

## Structure
- `controllers/` — Backend logic
- `models/` — Data models
- `views/` — HTML templates
- `assets/` — CSS, JS, images
- `includes/` — Config, utility functions

## Setup
1. Import the SQL schema
2. Configure `includes/config.php`
3. Place images in `assets/images/`
4. Run on XAMPP

## API Endpoints
See `ajax_actions.php` and `controllers/` for details.
